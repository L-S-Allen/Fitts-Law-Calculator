Fitts' Law Calculator

Fitts' Law calculates \(MT \), the time taken for a user to perform a given movement and reach the target object of that movement.

Where \(a \) and \(b \) are constants based on existing knowledge about the movement. \(D\) is the distance between the start point and the target, and \(W\) is the width of the target.

This calculator allows you to choose one of three versions of Fitts' law, input the values you have determined for the variables and calculate MT.

Choose a formula

$$MT = a + b \cdot \log_{2}({D \over W} + 1).$$
$$MT = a + b \cdot \log_{2}({D \over W}).$$
$$MT = a + b \cdot \log_{2}(2{D \over W}).$$
